Handwritten Digit Recognition Project Report

By Tanishq, Adwith, Srijith, Venu, Srikar, Arjun

Abstract

This report explores various machine learning models for handwritten digit recognition using the MNIST dataset. We evaluate Support Vector Machine (SVM), K-Nearest Neighbors (KNN), Gaussian Naive Bayes, Linear Regression, and Decision Tree based on accuracy and interpretability. The goal is to compare their strengths and weaknesses and guide future improvements.

Models Used & How They Work

1. Support Vector Machine (SVM)

Uses hyperplanes in high-dimensional space to separate digit classes. The RBF kernel helps handle non-linear data.

2. K-Nearest Neighbors (KNN)

Classifies digits by majority vote among the 'k' closest training examples in feature space.

3. Gaussian Naive Bayes

Assumes features are normally distributed and independent. Fast and simple, but can underperform on complex datasets.

4. Linear Regression

Applies regression to classify digits by predicting continuous values and selecting the closest class.

5. Decision Tree

Builds a flowchart-like structure to make decisions based on feature thresholds. Easy to interpret but may overfit.
